1. 설치

Mac

# 터미널(⌘+Space → 터미널) 열고 붙여넣기
$ curl -fsSL https://claude.ai/install.sh | bash
# 터미널 껐다 켜고
$ claude

Windows

# 먼저 Git 설치: git-scm.com/downloads/win
# PowerShell(파란 창) 열고 붙여넣기
> irm https://claude.ai/install.ps1 | iex
# 껐다 켜고
> claude

VS Code 확장

$ node --version # v18 이상 확인
$ npm install -g @anthropic-ai/claude-code
# VS Code → Extensions(⌘⇧X) → "Claude Code" 검색 → 설치

2. 자주 쓰는 명령어

  • /compact 대화 내용 압축 (토큰 절약)
  • /model sonnet 모델 전환 (비용 절약)
  • /status 현재 컨텍스트 사용량 확인
  • /cost 세션 비용 확인
  • claude doctor 문제 자동 진단

3. 첫 번째 스킬 만들기

Step 1 — 폴더 만들기

$ mkdir -p .claude/commands

Step 2 — 마크다운 파일 작성

.claude/commands/my-first-skill.md 파일을 만들고 업무 절차를 자연어로 작성

# 예시: 주간 보고서 자동화
1. 이번 주 작업 폴더에서 변경된 파일 목록을 확인한다
2. 각 파일의 변경 내용을 요약한다
3. 주간 보고서 양식에 맞춰 정리한다
4. weekly-report.md 파일로 저장한다

Step 3 — 실행

$ claude
> /my-first-skill

4. MCP 서버 연동

Apple 미리알림 연동 예시

$ claude mcp add apple-reminders -- npx -y @anthropic/apple-reminders-mcp
$ claude mcp list # 연결 확인

연동 후 "오늘 미리알림 보여줘", "내일까지 할 일 추가해줘" 등 자연어로 사용

5. 토큰 절약 팁

  • 범위 좁히기 — "이 프로젝트 개선해줘" ✗ → "src/auth.ts의 login 함수 수정해줘" ✓
  • 주제별 세션 분리 — 한 세션에서 모든 걸 하지 말고, 주제가 바뀌면 새 대화
  • 모델 전환 — 평소엔 Sonnet, 복잡한 작업만 Opus
  • /compact — 대화가 길어지면 과거 내용을 요약 압축

6. STAR 프레임워크 — 내 AI 활용 계획

S — Start (시작할 것)

오랫동안 미뤄온 업무 자동화가 있나요?

T — Try (도전할 것)

개발자에게 맡겨야 한다고 생각했던 것은?

A — Amplify (증폭할 것)

내가 이미 잘하는 것 + AI = ?

R — Recover (탐색할 것)

시간이 생기면 하고 싶었던 것은?
