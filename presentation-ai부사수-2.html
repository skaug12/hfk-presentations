오픈 클로, 이게 뭐야?

클로드 코드를 누구나 쓸 수 있도록 열어둔 기능

1회차에서 소개했던 클로드 코드의 핵심 기능들을 이제 여러분의 컴퓨터에서 직접 실행해볼 차례임.

오늘 세션이 끝나면 각자 컴퓨터에 클로드 코드가 설치되어 있고, 자연어로 명령을 내려볼 수 있는 상태가 되는 것이 목표임.

오늘의 흐름

지난 2주 돌아보기 → 프레임워크 배우기 → 설치 → 실습

클로드 코드로 할 수 있는 것들

  • 내 컴퓨터 파일 정리, 이름 바꾸기, 분류
  • 문서 작성 — 보고서, 이메일, 후기
  • 데이터 분석 — 엑셀, CSV 파일 처리
  • 웹사이트 만들기 — HTML, 발표 자료
  • 반복 작업 자동화 — 스킬로 저장

시작일

오늘부터. 세션 끝나면 바로 쓸 수 있음.

Review

2주 동안 나눈 이야기

1회차 이후, 멤버들과 주고받은 질문과 생각들.

자주 나온 질문들

  • "코딩 진짜 몰라도 되나요?"
  • "ChatGPT랑 뭐가 다른 건가요?"
  • "뭐부터 시켜봐야 하나요?"
  • "유료인가요? 얼마인가요?"
  • "혼자서도 설치할 수 있나요?"

공통된 고민

  • AI 도구가 너무 많아서 뭘 써야 할지 모르겠음
  • 배워봤자 또 새로운 게 나올 것 같음
  • 업무에 실제로 적용할 수 있을까 불안함
  • AI에 밀리지 않으려면 뭘 해야 할까

핵심 — 도구 자체보다 "내가 뭘 하고 싶은지"를 아는 게 먼저임. 오늘은 그 출발점을 만드는 시간.

Perspective

AI부사수 이후, 우리가 키워야 할 역량

"AI로 생산성 20배 올렸더니, 사람이 한 명도 안 남았다" — 심성재

AI가 잘하는 것

  • 빠르게 정보를 요약하고 정리
  • 반복 작업을 실수 없이 처리
  • 코드, 문서, 데이터를 대량으로 다루기
  • 패턴을 찾아서 분석

사람만 할 수 있는 것

  • 문제를 정의하는 능력 — 뭘 해결해야 하는지 아는 것
  • 맥락을 읽는 감각 — 상황에 맞게 판단하는 것
  • 관계를 만드는 힘 — 사람과 사람을 잇는 것
  • 의미를 부여하는 것 — 왜 이 일을 하는지

AI 연습도 중요하지만, 다시 관심 가져야 할 것이 있음.
어쩌면 우리는 무한루프로 생긴 매슬로의 욕구에 갇힌 건 아닐까 — 도구를 더 잘 쓰려는 욕구가 끝없이 쌓이지만, 정작 "왜 쓰는지"를 잊게 되는 것.

Framework

STAR 프레임워크

AI에게 일을 시킬 때, 이 4가지만 정리하면 결과가 달라짐.

S

Situation

상황

지금 어떤 상태인지

T

Task

과제

뭘 해야 하는지

A

Action

행동

어떻게 처리할지

R

Result

결과

어떤 결과물이 나와야 하는지

왜 STAR인가?

AI는 맥락을 모름. 내가 알려줘야 함. STAR로 정리하면 AI가 내 의도를 정확히 이해함.

ChatGPT에서도 통함

클로드 코드뿐 아니라 어떤 AI 도구에든 적용 가능한 범용 프레임워크임.

Practice

STAR 프레임워크 따라해보기

하나의 요청을 STAR로 분해해서 연습해봄.

Before — 그냥 시키기

> 팀 회의록 정리해줘

AI는 "어떤 회의? 어떤 형식? 누구한테 보낼 건데?"를 모름.

After — STAR로 시키기

# S: 상황
> 오늘 팀 회의 녹음 파일이 있어.

# T: 과제
> 핵심 논의 사항과 액션 아이템을
  정리해줘.

# A: 행동
> 안건별로 구분하고, 담당자를
  표시해줘.

# R: 결과
> Slack에 공유할 수 있게 마크다운
  형식으로 만들어줘.

지금 해보기

각자 업무에서 반복하는 일 하나를 떠올리고, STAR로 분해해봄.

  1. S — 지금 상황은?
  2. T — 뭘 해야 하는 건지?
  3. A — 어떻게 처리하면 좋은지?
  4. R — 최종 결과물은 어떤 형태인지?

팁 — 완벽하지 않아도 됨. 대충이라도 4개를 채우면, AI가 나머지를 물어봄.

Setup — Step 1

클로드 코드 설치하기

1

Claude 계정 만들기

claude.ai에 접속해서 회원가입

Google 계정으로 간편 가입 가능

2

Claude Code 구독하기

claude.ai/code에서 구독 시작

Pro 플랜: 월 $20 (약 27,000원)

Max 플랜: 월 $100 — 더 많이 쓸 수 있음

3

설치하기

구독 후 안내에 따라 설치

# Mac 터미널에서
$ npm install -g @anthropic-ai/claude-code

설치가 어렵다면

데스크톱 앱으로도 사용 가능함.

  • Mac 앱 — claude.ai/download에서 다운로드
  • 웹 앱 — claude.ai/code에서 바로 사용
  • VS Code 확장 — 다음 슬라이드에서 설명

터미널 설치는 고급 사용자용. 오늘은 VS Code 확장으로 진행함.

같이 하니까 괜찮음

막히는 곳은 바로 도와줄 것임. 그냥 따라오기만 하면 됨.

Setup — Step 2

Visual Studio Code 설치하기

클로드 코드를 편하게 쓸 수 있는 무료 에디터임. 개발자 도구지만, 우리는 AI 명령창으로 사용함.

1

VS Code 다운로드

code.visualstudio.com 접속

내 컴퓨터(Mac/Windows)에 맞는 버전 다운로드 → 설치

2

한국어 팩 설치 (선택)

확장(Extensions) 탭에서 "Korean" 검색

메뉴가 한국어로 바뀜. 안 해도 괜찮음.

3

작업 폴더 열기

File → Open Folder → 내 작업 폴더 선택

바탕화면에 "AI연습" 같은 폴더를 하나 만들어두면 좋음.

VS Code가 뭔지 몰라도 괜찮음

개발자들이 코드를 쓰는 프로그램인데,
우리한테는 그냥 "AI한테 말 거는 창"임.

메모장처럼 생겼는데, 여기서 클로드 코드를 부르면 내 파일을 직접 보고 작업해줌.

Word가 문서 작성용이라면,
VS Code는 AI 작업용이라고 생각하면 됨.

Setup — Step 3

클로드 코드 연결하기

VS Code에 클로드 코드 확장을 설치하면, 에디터 안에서 바로 AI를 부를 수 있음.

1

확장(Extensions) 열기

VS Code 왼쪽 사이드바에서 블록 모양 아이콘 클릭

또는 단축키: Mac Cmd+Shift+X / Win Ctrl+Shift+X

2

"Claude Code" 검색 → 설치

Anthropic 공식 확장을 찾아서 Install 클릭

설치 후 왼쪽 사이드바에 Claude 아이콘이 생김

3

Claude 계정 로그인

Claude 아이콘 클릭 → 로그인 안내를 따라 계정 연결

브라우저가 열리면 로그인하고 돌아오면 됨

연결 확인하기

Claude 패널이 열리면 성공임.

# Claude 채팅 창에 입력해봄

> 안녕, 잘 연결됐어?

# Claude가 대답하면 성공!

여기까지 되면 준비 완료 — 이제 자연어로 뭐든 시켜볼 수 있음.

Practice

자연어로 명령해보기

설치가 끝났으면, 이제 직접 해봄. 한국어로 편하게 말하면 됨.

Level 1 — 간단한 대화

> 자기소개 해줘

> STAR 프레임워크가 뭐야?

> 오늘 저녁 메뉴 추천해줘

ChatGPT처럼 대화하는 것. 여기까지는 똑같음.

Level 2 — 파일 만들기

> "hello.txt" 파일을 만들고
  "안녕하세요"라고 적어줘

> 이 폴더에 있는 파일 목록을
  알려줘

여기서부터 ChatGPT와 다름. 내 컴퓨터에 직접 실행함.

Level 3 — 업무 활용

> 이 폴더의 파일 이름을
  날짜_제목 형식으로 바꿔줘

> 내 이력서를 STAR 프레임워크로
  다시 정리해줘

이게 클로드 코드의 진짜 힘임.

오늘의 과제

각자 STAR로 정리한 요청을 클로드 코드에 입력해봄.

잘 안 되면 다시 고쳐서 시켜봄.

"잘 시키는 법"은 해봐야 늘음.
