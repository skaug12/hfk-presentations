클로드 코드를 누구나 쓸 수 있도록 열어둔 기능

1회차에서 소개했던 클로드 코드의 핵심 기능들을 이제 여러분의 컴퓨터에서 직접 실행해볼 차례임.

오늘 세션이 끝나면 각자 컴퓨터에 클로드 코드가 설치되어 있고, 자연어로 명령을 내려볼 수 있는 상태가 되는 것이 목표임.

오늘의 흐름

지난 2주 돌아보기 → 프레임워크 배우기 → 설치 → 실습

클로드 코드로 할 수 있는 것들

  • 내 컴퓨터 파일 정리, 이름 바꾸기, 분류
  • 문서 작성 — 보고서, 이메일, 후기
  • 데이터 분석 — 엑셀, CSV 파일 처리
  • 웹사이트 만들기 — HTML, 발표 자료
  • 반복 작업 자동화 — 스킬로 저장

오늘부터. 세션 끝나면 바로 쓸 수 있음.

Review — Q&A 1

카드뉴스 제작, 클로드 코드로 달라질 수 있을까?

멤버 질문

카드뉴스 업무 프로세스가 책 → ChatGPT/Gemini → Claude 레이아웃 → Canva 실제작 3단계인데, 의뢰측에서 "딸깍해서 대량 생산"을 원함.

근데 더 보고 싶은 콘텐츠로 만들려면 사람 터치가 필요하고, 적지 않은 시간이 듦. 업무를 아직 최적화 못한건지, 기능이 못 따라오는 건지 헷갈림.

핵심 질문 — "완성도 높은 콘텐츠"와 "대량 생산" 사이에서 클로드 코드가 도울 수 있는 건 뭔가?

내 답변

완성도 = GPT 티가 안 나게 + 나만의 콘텐츠 강점 반영이라면:

  1. 콘텐츠 소스를 PDF로 확보 — 클로드 코드가 언제든 꺼내 쓸 수 있게
  2. 지금까지 작업한 모든 텍스트 · 디자인 아웃풋 · 가이드를 백업
  3. 클로드 코드가 참고할 나만의 가이드 작성 — 과거 아웃풋 + 가이드를 모두 훑어본 뒤 새로운 결과물 생성
  4. Canva → Figma로 전환하면, Claude Code ↔ Figma 연동으로 텍스트 교체까지 한 큐에 가능

초안 만족도 40% → 70% 수준까지 상승 가능. 같은 템플릿을 쓴다면 Figma에서 글자 수만 지정해두면 됨.

Review — Q&A 2

더 나온 질문들

멤버 질문

처음 시작할 때 참고한 영상이나 자료가 있나요?

답변

영상, 자료를 따로 보지 않았음. 다른 사람들의 케이스를 많이 보고, 클로드 코드 맥스를 결제한 뒤 시행착오를 최대한 많이 쌓는 것을 추천.

$20 플랜도 좋지만, $100 플랜이 더 공부됨 — 많이 써봐야 감이 잡힘.

멤버 질문

회사 데이터를 클로드에 입력 못하는데, 어떻게 활용하나요?

"식약처 API 생산실적을 엑셀화하는 법, 수십만 셀 식품 정보 MI 분석, 웹에서 데이터 로우 추출·가공"

답변 — 실제 시연

프롬프트 9개(총 324자)로 파이프라인 완성:

Python 스크립트 2개 + 더미 데이터 5,000건 + HTML 대시보드 + GitHub Pages 배포 + 문서 3개

외부 API 데이터도 클로드 코드가 로컬에서 처리 — 보안 걱정 없음

공통 교훈 — 도구 자체보다 "내가 뭘 하고 싶은지"를 아는 게 먼저임. 오늘은 그 출발점을 만드는 시간.

Perspective

AI부사수 이후, 우리가 키워야 할 역량

"AI로 생산성 20배 올렸더니, 사람이 한 명도 안 남았다" — 심성재

AI가 잘하는 것

  • 빠르게 정보를 요약하고 정리
  • 반복 작업을 실수 없이 처리
  • 코드, 문서, 데이터를 대량으로 다루기
  • 패턴을 찾아서 분석

사람만 할 수 있는 것

  • 문제를 정의하는 능력 — 뭘 해결해야 하는지
  • 맥락을 읽는 감각 — 상황에 맞게 판단
  • 관계를 만드는 힘 — 사람과 사람을 잇는 것
  • 의미를 부여하는 것 — 왜 이 일을 하는지

AI 연습도 중요하지만, 다시 관심 가져야 할 것이 있음. 어쩌면 우리는 무한루프로 생긴 매슬로의 욕구에 갇힌 건 아닐까 — 도구를 더 잘 쓰려는 욕구가 끝없이 쌓이지만, 정작 "왜 쓰는지"를 잊게 되는 것.

내 STAR 정리해보기

각자 적어보기

업무든 개인이든, 떠오르는 것을 하나씩 적어봄.

  1. S — 미뤄왔던 건 뭐가 있지?
  2. T — 못한다고 생각했던 건?
  3. A — 이미 잘하는데 더 잘하고 싶은 건?
  4. R — 반복해서 귀찮은 작업은?

팁 — 완벽하지 않아도 됨. 대충이라도 4개를 채우면, AI가 나머지를 물어봄.

Before — 그냥 시키기

> 팀 회의록 정리해줘

AI는 "어떤 회의? 어떤 형식?"을 모름.

After — STAR로 시키기

# S: 미뤄왔던 일
> 회의 후 항상 정리를 미룸
# T: 못하던 것
> 요약 정리가 잘 안 됨
# A: 더 잘하고 싶은 것
> 액션 아이템 추출은 잘함, 더 구조화
# R: 반복 업무
> 매주 같은 형식으로 Slack에 공유

Level 2 — 파일 만들기

> 이 폴더의 파일 이름을

이게 클로드 코드의 진짜 힘임.
